# Equipment Maintenance Cost Analysis

## Objective
The objective of this project is to analyze equipment maintenance cost and downtime
to identify high-cost equipment and support data-driven maintenance decisions.

## Problem Statement
Unplanned breakdowns increase downtime and maintenance cost.
This analysis helps identify patterns and improvement areas.

## Dataset
Sample maintenance data including:
- Equipment name
- Month
- Maintenance cost
- Downtime hours
- Breakdown status

## Tools & Technologies
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Maintenance cost by equipment
- Breakdown frequency analysis
- Downtime trend analysis
- Cost comparison across equipment

## Key Insights
- Certain equipment shows higher maintenance cost
- Breakdown-prone equipment causes more downtime
- Preventive maintenance can reduce cost

## Business Impact
This analysis helps:
- Reduce unplanned downtime
- Optimize maintenance planning
- Improve equipment availability
